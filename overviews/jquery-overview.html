Règle d'or

Afin d'attendre le chargement total du DOM il faut toujours écrire sont code jQuery/JavaScript à l'intérieur de la fonction suivante : 

$(function () {
    // je dois absolument écrire mon code ici !
});

Les sélecteurs

Un sélecteur permet de récupérer un ou plusieurs éléments du DOM. Pour séléctionner un élément on utilise la fonction $(). Celle-ci prend en paramètre un sélècteur CSS valide (eg. #foo, .bar, etc.). 

// sélectionne tous les éléments avec l'id "foo"
var fooElements = $("#foo");
